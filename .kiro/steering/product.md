# Product Overview

Schedule TW Client é a aplicação mobile-first voltada para clientes finais realizarem agendamentos de serviços.

## Core Features

- Fluxo de agendamento em etapas (wizard):
  - Seleção de serviço
  - Escolha de serviços adicionais (opcionais)
  - Seleção de profissional (ou "sem preferência")
  - Escolha de data e horário disponível
  - Confirmação do agendamento
- Autenticação via telefone
- Visualização de agendamentos (próximos e histórico)
- Cancelamento de agendamentos com justificativa
- Integração com calendário (adicionar à agenda)
- Sistema de referência via URL (shopId e unitId codificados)

## User Experience

- Interface mobile-first otimizada para smartphones
- Fluxo guiado passo a passo
- Textos configuráveis via JSON
- Suporte a múltiplas unidades de um mesmo estabelecimento

## User Roles

O sistema é voltado para clientes finais que desejam agendar serviços em estabelecimentos cadastrados.

## Language

A aplicação está em português (Brasil).
